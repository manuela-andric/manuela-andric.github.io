Stability analysis and 2nd order theory on variable stiffness beams

Calculation of variable stiffness beams is explained in theoretical and numerical terms using the Python programming language for a console-based steel beam of varying stiffness. The task was solved using both the differential method and the transfer matrix method. The differential method was used for a linearly varying stiffness beam and the transfer matrix method was used for the beam in which each segment has constant stiffness calculated based on the average segment stiffness. The variable stiffness beam was divided on three, six and twelve segments to determinate the accuracy of the solution by increasing the number of segments. Both methods proved to be reliable with accuracy increase proportional to number of segments/divisions.

The analysis of hydraulic parameters during hydraulic (water) jump

This paper describes the process of creating hydraulic (water) jumps, its types, and channel protection measures by using various hydraulic engineering constructions that dissipate water jets, thus reducing the devastating effect of water jumps on the crust. In order to better understand the hydraulic leap, the work began with the description of the basic elements and the physical flow processes in the gentle sloping channels with stationary and uniform flow and rapid velocities.
